\section{Abstract}
\hspace{1em} En este trabajo se diseñará y construirá un pequeño coche 
teledirigido, con el objetivo de que pueda ser controlado a través de
cualquier telefono sin necesidad de instalar una aplicación,
utilizando para ello el navegador web del dispositivo.

Para ello se utilizará el microcontrolador ESP32, que cuenta con las 
capacidades necesarias para crear un servidor web al que conectarse
y enviar las señales de control al coche.

Para la construcción del coche, se utilizara la impresión 3D, cuyos materiales 
serán el PLA para la estructura principal y resina transparente para
la carroceria, con el fin de poder mostrar la electrónica interna del 
coche a los más curiosos.

En cuanto a la parte electrónica, se utilizarán motores DC para el movimiento
del coche, controlados por un driver de motores, y celulas recargables para
alimentar el sistema. Además, se utilizará un motor servo para controlar 
la dirección del coche.
